Commands :
  • GET TICKET - To get the details of a specific ticket using voice command Example - To get the details of ticket number 100, use the command - Get Ticket 100.
  • Priority Ticket - To get assigned tickets in a specific priority using voice command Example - To get high priority tickets - High Priority Ticket
  • REPLY WITH “Your text to add in reply” - To reply to a ticket using voice command Example - To reply to ticket number 100, use command - Reply 100 With I am replying to ticket 100 .
  • PUBLIC/PRIVATE NOTE WITH “Your text to add to note” - To add a note to a ticket using voice command Example - To add a public note to ticket number 100, use command - Public Note 100 With I am adding public note to ticket 100

    • Note : Text after with will be addes as reply/note
