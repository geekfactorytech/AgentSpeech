After recording the voice click 'apply' to add the text in editor window.
Note : The recorded text will be added in editor window of same ticket.
Record Apply
Commands :
  • Get specific ticket using : 'get ticket ticketnumber' eg : get ticket 10.
  • Get priority based tickets using : priority name priority tickets eg: high priority ticket.
  • Make a reply to ticket using : reply ticketnumber with "text you want to add reply" eg : reply 10 with This is a sample reply.
  • Add a note to ticket using : public/private note ticketnumber with "text you want to add reply" eg : public note 10 with This is a sample note.

    • Note : Text after with will be addes as reply/note
